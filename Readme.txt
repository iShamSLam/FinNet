# HyperLedger based decentralized network

# Requirements: 
1) Docker
2) Golang
3) Modified Binaries (can be fount at this repo)
4) Config tools (can be fount at this repo)

***
Installation
**
At the begining you need to start network via:
```
docker-compose -f docker-compose-fin-net.yaml -d

```
When you need to install and approve specified chaincode for each channel in the network:
Tutorials about it can be found at : hyperledger-fabric.readthedocs.io

Chaincode API description can be found at the chaincode folder of this repository
